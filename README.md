# pypull
<strong><b>Pcap tool written in Python using Scapy and the ipinfo API module.</b></strong>
# Requirements - Tested on Ubuntu LTS 18.04
<ul><strong><b>
 <li>Python3</li>
 <li>python3-pip</li>
 <li>scapy</li>
 <li>ipinfo</li>
  
</ul></b></strong>

# Installation
<strong><b>
 apt install python3
 apt install python3-pip
 pip3 install scapy
 pip3 install ipinfo
</b></strong>
