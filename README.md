<h1>PyPull!</h1>
<h2>Pcap tool written in Python3 using Scapy and the ipinfo API module.</h2>
<h1>Requirements(Installed by setup.sh) - Tested on Ubuntu LTS 18.04</h1>
<ul><strong><b>
 <li>root/sudo permissions for scapy</li>
 <li>python3</li>
 <li>python3-pip</li>
 <li>scapy</li>
 <li>ipinfo</li>
</ul></b></strong>

<h1>Installation</h1>
<strong>
 
 ```sh setup.sh```

</strong>

<h1>Usage</h1> 
<b>You can use any file you want containing <u><em>proper</em></u> bpf syntax with this script!</b>

<strong>

```pull filter.txt```

</strong>
