# pypull
<strong><b>Pcap tool written in Python using Scapy and the ipinfo API module.</b></strong>
# Requirements - Tested on Ubuntu LTS 18.04
<strong><b>
 Python3
 python3-pip
 scapy
 ipinfo 
  
</b></strong>
# Installation
<strong><b>
 apt install python3
 apt install python3-pip
 pip3 install scapy
 pip3 install ipinfo
</b></strong>
